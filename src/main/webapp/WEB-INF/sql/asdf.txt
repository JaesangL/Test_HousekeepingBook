 로그인 버튼 클릭시 아이디와 비밀번호는 아래와 같이 유효성 검사를 진행한다. 
 
 
 아이디를 입력하지 않았을 시 “아이디를 입력해 주세요” 문구를 alert로 출력 
 아이디가 3~8글자 사이가 아닐 시에는 “아이디는 3~8글자로 입력해 주세요” 
 문구를 alert로 출력 
 
 
 비밀번호를 입력하시 않았을 시 “비밀번호를 입력해 주세요” 
 문구를 alert로 출력 
 비밀번호가 5~10글자 사이가 아닐 시에는
  “비밀번호는 5~10글자로 입력해 주세요” 문구를 alert로 출력 
  
  
  
  
  - 로그인 버튼 클릭후, 로그인 성공시 
  세션에 유저 아이디를 저장하고 
  
  메인화면으로 이동 
  
  - 로그인 버튼 클릭후, 로그인 실패시 로그인 화면으로 이동 
 
 
 
 
 
 
 
 
 
 









가입 버튼 클릭시,
 아래와 같이 유효성 검사를 진행한다. 
 아이디를 입력하지 않았을 시 “아이디를 입력해 주세요” 
 문구를 alert로 출력 
 아이디가 3~8글자가 아닐 시 
 “아이디는 3~8글자로 입력해 주세요” 
 문구를 alert로 출력
  비밀번호를 입력하지 않았을 시 
  “비밀번호를 입력해 주세요” 문구를
   alert로 출력 
   
   
   비밀번호가 5~10글자가 아닐 시 “비밀번호는 5~10글자로 입력해 주세요” 
   문구를 alert로 출력
    비밀번호 항목과 비밀번호 확인 항목의 값이 동일하지 않을 시
     “동일한 비밀번호를 입력해 주세요” 문구를 
     alert로 출력 이름을 입력하지 않았을 시
      “이름을 입력해 주세요” 문구를 alert로 출력 
 
- 가입 버튼 클릭후, 회원가입 성공시 로그인 화면으로 이동
 - 가입 버튼 클릭후, 회원가입 실패시 회원가입 화면으로 이동
